A modular PowerShell toolkit for configuring, auditing, and hardening Windows 10 and Windows 11.

View on GitHub

🚀 What is Baseline?

Baseline is a manifest-driven Windows configuration utility with a full WPF GUI, compliance tracking, audit trails, presets, and headless automation support.

🔥 Key Features

  • GUI with search, filters, and risk labels
  • Preview-first execution with visual diff
  • Compliance tracking and drift detection
  • Headless automation and scheduled runs
  • Preset-driven configuration (Minimal → Advanced)
  • Audit logs and configuration snapshots

⚙️ Presets

  • Minimal – Safe starting point
  • Basic – Recommended for most users
  • Balanced – More advanced tuning
  • Advanced – Expert-level changes

▶️ Quick Start

.\run.cmd
    
.\Baseline.ps1 -Preset Basic
